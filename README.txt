# AMAPP - Adaptive Music Application

Welcome to AMAPP! This application helps composers and audio implementers streamline adaptive music workflows between REAPER and game audio middleware.

## Requirements

### REAPER Extensions (Required)
- SWS/S&M Extension: https://www.sws-extension.org/
- ReaImGui (Dear ImGui): Install via ReaPack
- js_ReaScriptAPI: Install via ReaPack

### For Wwise Integration (Optional)
To use the Wwise implementation features:
- Wwise must be running with a project open
- WAAPI (Wwise Authoring API) must be enabled in Wwise:
  Project > User Preferences > Enable Wwise Authoring API

Note: No additional software (Python, etc.) needs to be installed. AMAPP includes
all necessary components for Wwise communication.

## Installation

1. Install the required REAPER extensions listed above
2. Run the AMAPP installation script in REAPER
3. The AMAPP action will appear in REAPER's Actions menu

## Wwise Setup

1. Open Wwise with your project
2. Enable WAAPI: Project > User Preferences > Enable Wwise Authoring API
3. In AMAPP, use "Implementation Design" to export your adaptive music structure
4. Your music hierarchy, events, and soundbanks will be created automatically in Wwise
